Ordinality PASS
CV stability ≥ 0.5
Deterministic. Cannot be overridden.
Layer 2: LLM (GPT-4.1)Receives HARD + SOFT criteria
EDA findings (dynamic from data)
Convergence as ground truth
Must RETRY if HARD fails
Can trigger RETRY on SOFT issues
Layer 3: Decision LogicRule FAIL → RETRY (always)
Rule PASS + LLM RETRY → RETRY
Rule PASS + LLM ACCEPT → PROCEED
Max iterations → PROCEED
AGENT DECISION
✅ PROCEED → scenarios + narratives
🔁 RETRY → adjust strategy → loop back
🔁 RETRY: Iter 1→2: expand to group specs (G-J) · Iter 2→3: switch to monthly granularity
